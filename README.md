# README

How to run server  
git clone https://github.com/RobertoFalconi95/Kalypso  
cd Kalypso  
rails db:migrate  
rails db:seed  
rails s  

How to run tests  

git clone https://github.com/RobertoFalconi95/Kalypso  
cd Kalypso  
rails db:migrate  
rails db:seed  
bundle exec cucumber  
bundle exec rspec
